# M1-CH1 Robot

### Components:

- Arduino Mega2560.
- Protoboard.
- Servomotor SG90 (x5).
- Sensitive touch.
- Ultrasonic HC-SR04.
- Camera OV7670.
- Old smartphone vibration motor (3.3v aprox).
- 18650 battery cell (3.7v 2a) (x2), connected in series. Power supply for servomotors.
- LM2596 Step down.
- 9v Battery cell (0.2a). Power supply for arduino.
